# Road trip Écosse · carte interactive

Carte de voyage interactive (Écosse, 10 jours) — un seul fichier HTML autonome,
vanilla JS + SVG, zéro dépendance runtime. Carte dessinée depuis un contour
GeoJSON projeté, photos incrustées en base64, pan/zoom tactile, fiches détaillées.

## Voir le site

Publié via GitHub Pages : **https://vivienadnot.github.io/road-trip-scotland/**

Ou ouvrir `index.html` localement dans un navigateur.

## Structure

- `index.html` — l'application complète, auto-contenue (géométrie + données + photos inline).
- `pictures/` — photos sources (Wikimedia Commons, CC/CC0 — créditer les auteurs).
- `HANDOFF.md` — contexte projet, architecture interne, roadmap (FR).

Voir `HANDOFF.md` pour l'architecture, les contraintes apprises et la suite
(carte Irlande, itinéraire jour/jour, budget).
